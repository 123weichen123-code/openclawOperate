# Game Analyst

游戏数据分析技能，用于抓取热门游戏数据并生成市场分析报告。

## 支持的游戏类型

- **FPS射击游戏**: 无畏契约(Valorant)、CS2、Apex英雄等
- **卡牌游戏**: 杀戮尖塔(Slay the Spire)、游戏王、炉石传说等
- **MOBA**: 英雄联盟、DOTA2等
- **主机/3A**: 艾尔登法环、博德之门3等

## 数据来源

优先使用以下来源获取游戏数据：
- Steam商店页面
- 腾讯视频游戏中心
- 百度贴吧/知乎游戏区
- YouTube游戏频道
-  Twitch/Discord统计数据

## 分析维度

### 市场分析报告包含

1. **热度分析**
   - 搜索引擎热度趋势
   - 社交媒体讨论量
   - 直播平台观看人数

2. **用户评价**
   - 各平台评分对比
   - 好评/差评关键点

3. **竞品对比**
   - 同类型游戏对比
   - 市场份额变化

4. **趋势预测**
   - 未来更新预期
   - 市场走向分析

## 输出格式

```markdown
# [游戏名] 市场分析报告

## 📊 热度数据
[数据]

## ⭐ 用户评价
[评价摘要]

## 🔄 竞品对比
[对比分析]

## 📈 趋势预测
[预测]
```

## 注意事项

- 优先使用web_fetch获取数据，失败则尝试browser工具
- 如果数据源被拦截，记录问题并尝试其他来源
- 报告要基于实际数据，避免主观臆断
